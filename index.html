Projects

iTunes Preview App

This project is a simple app that allows users to search for music on iTunes and listen to a 30-second sample of each song. The app has a user-friendly interface that makes it easy for users to enter their search criteria and browse through the results. When a user finds a song they are interested in, they can click on it to listen to a 30-second sample. This feature allows users to get a sense of the song before deciding whether to purchase it or add it to their playlist. The app is a great tool for music lovers looking to discover new artists and tracks.

Pokemon List App

This project is a web application built using React.js that fetches a database of Pokémon and displays a list of them. The app allows users to browse through the list of Pokémon and view additional information about each one, such as their type, abilities, and stats. The app also includes a search feature that allows users to find specific Pokémon by number.

Space Invaders Clone

This project is a desktop space invaders clone built using JavaScript and the canvas element. It is a fun and engaging game that is inspired by the classic arcade game, but with modern graphics and gameplay mechanics. I mostly created this project in order to learn canvas with help of Chris Courses, but I changed most elements and added local storage, UI and difficulty on my own.

Movies Discovery App

The project is a web application built using React.js that helps users discover new movies.
